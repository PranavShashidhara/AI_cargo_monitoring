④ Risk scoring agent (RAG over FDA / GDP regulations)
Risk Scoring Agent
score_risk(state: CargoState) → CargoState
reads from state anomaly_type, severity, delay_hrs, stall_confirmed, hold_reason vaccine_type, pack_time, viability_window_hrs (from shipment record) RAG query ChromaDB ← FDA cold chain docs, GDP regulations, WHO vaccine guidelines writes to state spoilage_prob (0.0–1.0), action_window_hrs (remaining time before point-of-no-return) rag_citation, risk_narrative (LLM-generated plain english explanation)
⑤ Viability budget check
Viability Budget Check
check_viability_budget(state: CargoState) → CargoState
reads action_window_hrs, total_accumulated_delay_hrs spoilage_prob, route_options (alternate routes + time cost) computes delay_budget = action_window_hrs − total_accumulated_delay_hrs writes delay_budget_hrs, shipment_viable (bool) recommended_action, action_time_cost_hrs
⑥ Decision orchestrator — conditional routing node
Decision Orchestrator
orchestrate_decision(state: CargoState) → Command
reads anomaly_type, shipment_viable, delay_budget_hrs, recommended_action stall_confirmed, flight_event_type, hold_reason, sensor_silence routes to one of → NORMAL → continue_monitoring TEMP_BREACH + viable → cold_storage_intervention REEFER_FAILURE → emergency_vehicle_swap TRUCK_STALL → alternate_carrier_agent FLIGHT_CANCEL → flight_rebooking_agent CUSTOMS_HOLD → compliance_escalation_agent SENSOR_SILENCE → assume_breach_agent NOT viable (delay_budget < 0) → emergency_resupply_agent
NOMINAL TEMP_BREACH REEFER_FAILURE TRUCK_STALL FLIGHT_CANCEL CUSTOMS_HOLD NOT_VIABLE
