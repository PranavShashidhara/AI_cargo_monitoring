# AI_cargo_monitoring
AI cargo monitoring for pharma shipments using real-time IoT data. Detects anomalies, predicts spoilage/delay risks, and recommends rerouting, storage fixes, or customs actions. Triggers downstream updates (providers, inventory, logs) with human-in-the-loop and GDP/FDA compliance.

## Setup and Run Instructions

This project can be run either on your local machine using `setup.sh` or in a Docker container for a fully isolated environment.

---

## **1. Using `setup.sh` (Local Environment)**

### Run the script 
```bash
> bash setup.sh
```
If a virtual environment is created (usually venv), activate it:

```bash 
> source venv/bin/activate
```

## 2. Using Docker (Containerized Environment)

Build docker 
```bash 
> source venv/bin/activate
```
